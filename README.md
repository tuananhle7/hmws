# Continuous Memoised Wake Sleep

## Installation

```
git clone git@github.com:tuananhle7/cmws.git
cd cmws
pip install .
```
or
```
pip install -e .
```
for editable mode.

Install Sid's "rws" branch of pyro
```
pip install git+https://github.com/iffsid/pyro.git@rws
```

## Unit tests
```
pytest tests
```
